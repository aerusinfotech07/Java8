
# ✅ Core Java MCQs (51–100)

---

### **51. What is the default value of an object reference?**

A. 0
B. undefined
C. null
D. blank

**Answer:** C
**Explanation:** Object references are initialized to `null` by default.

---

### **52. Which of the following is not an exception?**

A. IOException
B. SQLException
C. Error
D. RuntimeException

**Answer:** C
**Explanation:** `Error` represents serious issues not meant to be handled like exceptions.

---

### **53. Which keyword is used to explicitly throw an exception?**

A. throws
B. throw
C. catch
D. try

**Answer:** B
**Explanation:** `throw` is used to create and throw an exception.

---

### **54. Which block handles an exception?**

A. try
B. throw
C. catch
D. finally

**Answer:** C
**Explanation:** `catch` block handles exceptions.

---

### **55. Which collection is fastest for retrieval operations?**

A. LinkedList
B. ArrayList
C. HashSet
D. TreeSet

**Answer:** B
**Explanation:** ArrayList provides fast random access using index.

---

### **56. Which collection class is synchronized?**

A. ArrayList
B. HashMap
C. Vector
D. HashSet

**Answer:** C
**Explanation:** Vector is thread-safe (synchronized).

---

### **57. Which data structure follows LIFO?**

A. Queue
B. Stack
C. ArrayList
D. LinkedList

**Answer:** B
**Explanation:** Stack uses Last-In-First-Out.

---

### **58. Which data structure follows FIFO?**

A. Stack
B. Queue
C. Set
D. Map

**Answer:** B
**Explanation:** Queue follows First-In-First-Out.

---

### **59. Which method removes an element from a collection?**

A. delete()
B. remove()
C. pop()
D. clearOne()

**Answer:** B
**Explanation:** `remove()` deletes an element.

---

### **60. Which class is thread-safe for string operations?**

A. String
B. StringBuilder
C. StringBuffer
D. CharSequence

**Answer:** C
**Explanation:** StringBuffer is synchronized.

---

### **61. Which of the following is not a Java keyword?**

A. class
B. main
C. static
D. void

**Answer:** B
**Explanation:** `main` is a method name, not a keyword.

### **62. Which is not a primitive type?**

A. int
B. double
C. Object
D. char

**Explanation:** Object is a class.

### **63. Which class is parent of all exceptions and errors?**

A. Exception
B. Error
C. Throwable
D. Object

**Answer:** C
**Explanation:** `Throwable` is the root class.

### **64. What is runtime polymorphism?**

A. Method overloading
B. Method overriding
C. Constructor overloading
D. Compilation

**Answer:** B
**Explanation:** Achieved via method overriding.

---

### **65. What is compile-time polymorphism?**

A. Overriding
B. Overloading
C. Inheritance
D. Abstraction

**Answer:** B
**Explanation:** Method overloading is resolved at compile time.

### **66. Which annotation is used for method overriding?**
