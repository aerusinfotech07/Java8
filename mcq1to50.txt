
# ✅ Core Java MCQs (1–50)

---

### **1. Which of the following is not a Java feature?**

A. Object-oriented
B. Use of pointers
C. Portable
D. Dynamic

**Answer:** B
**Explanation:** Java avoids pointers to ensure security and simplicity.

---

### **2. What is JVM?**

A. Java Variable Machine
B. Java Virtual Machine
C. Java Verified Machine
D. Java Visual Machine

**Answer:** B
**Explanation:** JVM runs Java bytecode and provides platform independence.

---

### **3. Which keyword is used for inheritance?**

A. this
B. super
C. extends
D. implements

**Answer:** C
**Explanation:** `extends` allows one class to inherit another.

---

### **4. Which method is the entry point of a Java program?**

A. start()
B. run()
C. main()
D. init()

**Answer:** C
**Explanation:** Execution starts from `public static void main(String[] args)`.

---

### **5. Which of the following is not an OOP concept?**

A. Encapsulation
B. Polymorphism
C. Compilation
D. Inheritance

**Answer:** C
**Explanation:** Compilation is part of program execution, not OOP.

---

### **6. Which keyword prevents inheritance?**

A. static
B. final
C. abstract
D. private

**Answer:** B
**Explanation:** A `final` class cannot be extended.

---

### **7. Which is not a primitive data type?**

A. int
B. float
C. String
D. boolean

**Answer:** C
**Explanation:** `String` is a class, not a primitive.

---

### **8. What is the size of int in Java?**

A. 2 bytes
B. 4 bytes
C. 8 bytes
D. Depends on OS

**Answer:** B
**Explanation:** Java int is always 4 bytes (32 bits).

---

### **9. Which method is used to compare object contents?**

A. ==
B. equals()
C. compare()
D. instanceof

**Answer:** B
**Explanation:** `equals()` compares values/content, while `==` compares references.

---

### **10. Default value of boolean?**

A. true
B. false
C. null
D. 0

**Answer:** B
**Explanation:** Default boolean value is `false`.

---

### **11. Which loop executes at least once?**

A. for
B. while
C. do-while
D. foreach

**Answer:** C
**Explanation:** `do-while` runs before checking condition.

---

### **12. Which are exception handling keywords?**

A. try
B. catch
C. throw
D. All

**Answer:** D
**Explanation:** All are used in exception handling.

---

### **13. Superclass of all Java classes?**

A. Object
B. Class
C. System
D. String

**Answer:** A
**Explanation:** Every class inherits from `Object`.

---

### **14. Which package contains Scanner?**

A. java.io
B. java.util
C. java.lang
D. java.net

**Answer:** B
**Explanation:** Scanner is in `java.util`.

---

### **15. How to start a thread?**

A. run()
B. start()
C. init()
D. execute()

**Answer:** B
**Explanation:** `start()` creates a new thread and calls `run()` internally.

---

### **16. Which class is immutable?**

A. String
B. StringBuilder
C. ArrayList
D. HashMap

**Answer:** A
**Explanation:** String objects cannot be modified once created.

---

### **17. Which is a checked exception?**

A. NullPointerException
B. ArithmeticException
C. IOException
D. ArrayIndexOutOfBoundsException

**Explanation:** Checked exceptions must be handled at compile time.

### **19. Most restrictive access modifier?**
